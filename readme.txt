== Description ==

Responsive Latest News/Posts/Feeds/Articles is easy to use. You can add as many News/Posts/Feeds/Articles as you want. 
If you want to modify or want to add extra features, please contact the owner faferdaus@gmail.com.

= Uses =
Use this shortcode to show the News/Posts/Feeds/Articles [lnpa-latest-news]

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* send any data to external servers;
* use cookies.

== Installation ==

1. Upload the entire `Latest-News-Posts` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **LNPA Settings** menu in your WordPress admin screen.
